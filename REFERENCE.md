# Quick Stop · REFERENCE (external winners only — Blocklist Law)

Studied fresh for this build. No internal builds referenced.

## Award-grade references (vertical + adjacent, electric-night / signage / kinetic)
1. **Neon-signage editorial sites** (Awwwards nightlife/retail) — real neon tube
   construction with layered glow blooms and subtle flicker, not a PNG. Takeaway: build
   the OPEN sign from stroked SVG tubes + multiple drop-shadow glows + a flicker keyframe.
2. **Kinetic-typography marquee sites** (SiteInspire/Godly) — oversized condensed type on
   diagonal scrolling bands, speed reacting to scroll. Takeaway: diagonal ribbons of
   inventory words, scroll-velocity coupled, condensed Khand all-caps.
3. **Modern bodega / c-store rebrands** (Behance) — the best ones lean into "everything,
   all the time" with bold color blocks and a confident wordmark, never a stock-shelf
   carousel. Takeaway: lead with a CONFIDENT electric statement + a real product grid,
   not a generic "Welcome to our store."
4. **24/7 / always-on brand systems** — the always-on state IS the brand. Takeaway: wire
   the real hours into a live status the visitor can feel (the neon flips on schedule).

## Art direction
- Ground: near-black electric night `#0B0710` with magenta city-glow blooms.
- Accent: `#8E3B8E` magenta, lifted to a hot `#D464D4` for neon tube cores.
- Type: Khand (display, condensed signage) + Inter (body).
- Materiality: neon glass tubes, glow bloom, wet-asphalt sheen, diagonal ribbon bands.
- Motion: clock-wired neon flicker + scroll-velocity diagonal ribbons + reveal rises.
- Imagery: real night/convenience-store/coffee/ATM/lotto photography, magenta-graded,
  each used once, none shared with Orange siblings.
